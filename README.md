# morse

A simple Document Accessibility Checker

## Roadmap

-   Image Alt-text Checks
-   Font Checks
    -   Font sizing
    -   Font colours
    -   Font Weight
    -   Flesch-Kincaid
-   Headings
    -   Are headings in order and none missing?
-   Link Text Accessibility
    -   Is hyperlink text meaningful?
-   Table Accessibility
-   Form Accessibility
